    iid_estimate(samples, observable=None) ¶

    Estimate a scalar expectation from IID samples.

    observable is evaluated once on the full sample array and should return one scalar value per sample. When omitted, samples itself must be one-dimensional. The reported variance is the unbiased sample variance for n >= 2.

    estimate_normalization_ratio(log_weights) ¶

    Estimate a normalizing-constant ratio from importance log weights.

    If X ~ q and log_weights = log(tilde_p(X)) - log(q(X)), this estimates Z_p. More generally, when samples are drawn from the normalized version of tilde_q, the same calculation estimates Z_p / Z_q.

    The logarithm is always computed stably. The linear estimate is inf only when the result is too large to represent in float64.
