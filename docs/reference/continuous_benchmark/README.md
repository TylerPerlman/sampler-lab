# v0.12.0 Continuous Benchmark Reference

This directory is a fixed-seed **quick-budget** teaching and regression artifact generated by:

```bash
sampler-lab-benchmark \
  --quick \
  --replicates 2 \
  --figures \
  --seed 2022 \
  --output-dir docs/reference/continuous_benchmark
```

It covers all four exact-reference targets and every compatible adapter. The run produced 108
successful method/target/replicate results, two explicit reverse-KL multimodality exclusions, and no
runtime failures.

The small budget is intentionally suitable for repository review. It highlights qualitative failure
modes—mode trapping, funnel-neck bias, weight collapse, and approximate-particle mismatch—but is not
a precision leaderboard. Use larger `--samples`, `--warmup`, `--reference-samples`, and `--replicates`
for research comparisons.
